MicroAlg

Cette application vous permet d’exécuter du code MicroAlg sur votre machine.

Dépôt GitHub

Important : Pour l’instant, cette application ne permet pas d’écrire de code. Pour cela, il faut installer un éditeur de texte. Une fois le code écrit, vous pouvez le « partager » vers cette application ou ouvrir le fichier contenant le code.

MicroAlg est un langage de programmation en français dédié à l’algorithmique et à son enseignement. Simple (des parenthèses et des guillemets pour seuls caractères spéciaux), elle offre un cadre rigoureux pour l’apprentissage des concepts les plus importants de l’algorithmique et de la programmation impérative.

Pour plus d’informations, voir le site officiel de MicroAlg.
